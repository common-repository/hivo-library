== Description ==

This plugin allows users of HIVO to add Assets from their HIVO Library to the WordPress Media Library.

1. Login page

== Installation ==

1. Download and activate the plugin through the WordPress plugins screen.
2. Navigate to "HIVO Connector" in the admin panel.
3. Log in to HIVO wth your email and password.
4. After successfully logging in, use the "Library" tab at the top to search your HIVO library.
5. Click a version button below an Asset to add it to the WordPress Media Library.
